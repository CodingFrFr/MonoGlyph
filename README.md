# MonoGlyph

MonoGlyph is a small Python library for drawing ASCII/ANSI graphics with shading and rotation.

## Installation

```bash
git clone https://github.com/CodingFrFr/monoglyph
cd pythia
```
## Usage
```python
import MonoGlyph

mg = MonoGlyph(width=80, height=24)
mg.set_rotation(45, origin_x=40, origin_y=12)

# Stuff will be added to this
```
